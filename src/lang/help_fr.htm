Aide

Quand je démarre UMWP Autochanger, aucune fenêtre n'apparaît.
Si vous avez déjà enregistré des des sets, la fenêtre est automatiquement réduite dans la zone système, double-cliquez sur l'icône pour l'afficher (ou faites clic-droit => "Afficher"). Ce comportement peut être changé dans le menu "Options".
Puis-je renommer un set ?
Oui. Double-cliquez sur son nom, il vous sera demandé de choisir le nouveau nom.
Comment les raccourcis fonctionnent ?
Les raccourcis vous permettez de changer le set actif depuis le clavier même quand la fenêtre de l'application est masquée. Vous pouvez configurer les touches des raccourcis dans le menu "Options". Si par exemple vous avez choisi Maj+Ctrl, appuyez sur Maj+Ctrl+1 pour activer le premier set, Maj+Ctrl+2 pour le second set, etc. Seuls les 9 premiers sets peuvent être activés depuis le clavier.
Comment désactiver les raccourcis ?
Décochez toutes les touches dans le menu "Options".
Puis-je réordonner les sets ?
Oui. Simplement par glisser-déposer.
