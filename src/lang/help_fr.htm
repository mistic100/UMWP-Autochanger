Quand je démarre UMWP Autochanger, aucune fenêtre n'apparaît.
Si vous avez déjà enregistré des des sets, la fenêtre est automatiquement réduite dans la zone système, double-cliquez sur l'icône pour l'afficher (ou faites clic-droit => "Afficher"). Ce comportement peut être changé dans le menu "Options".
Puis-je éditer un set ?
Oui. Double-cliquez sur son nom, Vous pourrez choisir le nom, le type et le style de fond d'écran et le raccourci du set.
Comment les raccourcis fonctionnent ?
Il y a deux types de raccourcis ! ceux définis dans le menu "Options" permettent d'afficher/masquer l'application, refraîchir le fond d'écran et mettre pause/relancer le timer.
Ceux définis dans la boîte de dialogue d'édition des sets permettent de changer le set courant. Vous pouvez définir le même raccourci pour plusieurs sets.
Dans les deux cas, cliquez dans la boîte de texte et pressez la combinaison de touches désirée. Vous devez utiliser au moins un modifieur (Ctrl, Maj, Alt, Win) et certains touches sont interdites (Echap, Tab, Retour, ...). Pressez Echap ou Retour pour effacer le raccourci.
Puis-je réordonner les sets ?
Oui. Simplement par glisser-déposer.
