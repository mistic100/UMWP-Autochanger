Help

When I launch UMWP Autochanger, the window doesn't appear.
When you have already registered sets, the window automatically minimize to the system bar, double-click on the icon (or right-click => "Show") to open it. This behavior can be changed on the "Options" menu.
Can I edit a set ?
Yes. Double-click on it's name, you can set the name, the type and the style of wallpaper and the hotkey of the set.
How hotkeys work ?
There are two kind of hotkeys: the ones defined in the "Options" dialog allow to show/hide the application window, refresh the wallpaper and start/pause the timer.
The ones defined in the set edit dialog allow to change the current set. You can assign the same hotkey to multiple sets.
In both cases, click on the input box and press the desired keys combinination. You must at least use one modifier (Ctrl, Shift, Alt, Win) and some keys are not available (Esc, Tab, Backspace, ...). Press Esc or Backspace to clear the hotkey.
Can I reorder sets ?
Yes. Simply drag & drop them.
