Help

When I launch UMWP Autochanger, the window doesn't appear.
When you have already registered sets, the window automatically minimize to the system bar, double-click on the icon (or right-click => "Show") to open it. This behavior can be changed on the "Options" menu.
Can I edit a set ?
Yes. Double-click on it's name, you can set the name, the type and the style of wallpaper.
How hotkeys work ?
Hotkeys allows you two quickly change witch set is activated from the keyboard even when the application window is hidden. You can configure which keys must be pressed in the "Options" menu. Say you choosed hift+Ctrl, press Shift+Ctrl+1 to activate the first set, Shift+Ctrl+2 for the second set, and so on. Only the 9 first sets can be activated from the keyboard.
How can I unactivate hotkeys ?
Uncheck all keys in the "Options" menu.
Can I reorder sets ?
Yes. Simply drag & drop them.
